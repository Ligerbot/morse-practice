# Morse practice

Choose two different modes, one where you type dashes and dots and the program converts them, or one where you hold or tap space bar for variable length dashes and dots.

`python3 beeper.py` runs the converter from dots and dashes to sound.

playsound is required, otherwise on Linux you might hear weird screeching.

To add more of a gap between words just add extra spaces.

Also to enable or disable the auto id at start check in the top of beeper.pi for `autoid = True`
